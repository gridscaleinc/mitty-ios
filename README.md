# mitty-ios
Mitty iOS 版开发用工程

# CocoaPods

1. Install ruby

2. install cocoapods

```
$ gem install cocoapods
```

# Project

```
$ git clone git@github.com:gridscaleinc/mitty-ios.git
$ cd mitty-ios
$ pod install
$ open mitty.xcworkspace
```
